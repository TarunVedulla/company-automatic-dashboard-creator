# Company Automatic Dashboard Creator

Modern AI-powered company analysis dashboard.

## Features
- Interactive charts using Plotly.js
- AI summaries powered by Groq
- Background image + color customization
- Data persistence (remembers uploads)
- Real-time CSV auto-reload

## How to Run

1. Clone or download the project
2. Open `index.html` in your browser
3. Upload your CSV file
